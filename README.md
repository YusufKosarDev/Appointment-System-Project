# 📅 Appointment System Project

A clean and modern Appointment Booking System built with vanilla HTML, CSS, and JavaScript. It allows users to easily create and manage appointments through a simple and user-friendly interface, making it a great portfolio project for frontend development.


## 📝 What Does It Do?

The Appointment System provides a straightforward way to schedule and track appointments:

Create a new appointment by entering name, service, date, and time
Simple and intuitive user interface for quick booking
Display active appointments dynamically
Prevent empty or invalid form submissions
Clean and responsive design
Language switch option (TR / EN) (if active in your project)


## ⚙️ How Does It Work?

The project is built using fundamental web technologies:

HTML Structure

Form inputs for user data (name, service, date, time)
Appointment list section to display active bookings
Semantic and organized layout

CSS Styling

Modern card-based UI design
Responsive layout for different screen sizes
Styled buttons and form elements
Clean typography and spacing

Vanilla JavaScript

Form validation and event handling
Dynamic DOM updates for appointments
Handling user interactions (create, display)

Data Handling

Appointments are stored temporarily in JavaScript (or localStorage if you added it)
Updates UI instantly without page reload
